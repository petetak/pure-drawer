Setup

Head over to Github and download either the scss or css version or download all files. Further documentation below.

Markup

It is important that the markup is in the same order as below due to the use of the css sibling selector. If you add additional containers to your markup you need to update the css accordingly.

<div class="pure-container" data-effect="pure-effect-1" data-position="left">
    <input type="checkbox" id="pure-toggle" class="pure-toggle">
    <label class="pure-toggle-label" for="pure-toggle">+</label> 

    <div class="pure-drawer">
        Put your off canvas content here
    </div>  
    <div class="pure-pusher-container">
    	<div class="pure-pusher">
        	Put your site content here
        </div>
    	<label class="pure-overlay" for="pure-toggle"></label> 
	</div>
</div>

Attributes

data-effect: This is where you enter your desired effect. Ranges from 1-9.

data-position: This is where you enter your desired drawer position. Legal values are 'left', 'right' and 'top'.

SCSS Settings

// Toggler

$toggler-color: #d95843;
$toggler-checked-color: #fff;

// Drawer

$drawer-width-small: 100%;
$drawer-width-medium: 300px;
$drawer-width-large: 400px;

$drawer-height-small: 300px;
$drawer-height-medium: 100%;
$drawer-height-large: 100px;

$drawer-bgcolor: #d95843;

// Pusher

$pusher-bgcolor: #fff;

// Pusher Overlay

$overlay-bgcolor: rgba(0,0,0,0.4);

Breakpoints

Breakpoints are now included in the breakpoints.scss file. Set drawer width and height for small, medium and large screens.
